## Default Permission

Default permissions for the Terminal plugin

#### This default permission set includes the following:

- `allow-terminal-create`
- `allow-terminal-screenshot`
- `allow-terminal-send-keys`
- `allow-terminal-read-output`
- `allow-terminal-close`
- `allow-terminal-show-window`
- `allow-terminal-list-sessions`
