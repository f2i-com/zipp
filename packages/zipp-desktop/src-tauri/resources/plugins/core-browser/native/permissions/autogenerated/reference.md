## Default Permission

Default permissions for the Zipp Browser plugin

#### This default permission set includes the following:

- `allow-webview-create`
- `allow-webview-navigate`
- `allow-webview-evaluate`
- `allow-webview-action`
- `allow-webview-html-callback`
- `allow-webview-screenshot`
- `allow-webview-screenshot-callback`
- `allow-webview-close`
- `allow-webview-get-html`
