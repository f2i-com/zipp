## Default Permission

Default permissions for the Zipp Filesystem plugin

#### This default permission set includes the following:

- `allow-list-folder`
- `allow-read-file`
- `allow-write-file`
- `allow-pick-folder`
- `allow-pick-file`
- `allow-native-copy-file`
- `allow-calculate-file-chunks`
- `allow-read-chunk-content`
- `allow-get-downloads-path`
- `allow-get-app-data-dir`
- `allow-get-temp-dir`
- `allow-run-command`
